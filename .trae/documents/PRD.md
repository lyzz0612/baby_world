# 儿童动物认知应用 - 产品需求文档

## 1. 产品概述

一款专为平板设备设计的儿童教育应用，通过趣味互动方式帮助3-8岁儿童认识不同环境中的动物。

**核心价值：**
- 培养儿童对动物的认知和兴趣
- 通过声音互动增强学习体验
- 支持农场、动物园、海洋馆三大场景的探索

## 2. 核心功能

### 2.1 目标用户
- **主要用户**：3-8岁儿童
- **辅助用户**：家长/教师

### 2.2 功能模块

#### 首页模块
- 应用Logo和欢迎语
- 功能入口卡片：
  - 认识动物（主功能）
  - 更多功能（预留）

#### 认识动物页面
- **分类导航栏**：
  - 农场动物
  - 动物园动物
  - 海洋馆动物
- **动物卡片网格**：
  - 2列布局（平板适配）
  - 动物图片（可爱卡通风格）
  - 动物名称（中文大字）
- **点击交互**：
  - 显示动物图片
  - 播放语音："[动物名]怎么叫："
  - 播放动物叫声

## 3. 核心流程

```mermaid
graph TD
    A[打开应用] --> B[首页]
    B --> C[点击'认识动物']
    C --> D[选择分类]

    E --> F[展示动物列表]
    F --> G[点击动物卡片]
    G --> H[播放语音和动物叫声]

```

## 4. 用户界面设计

### 4.1 设计风格
- **主题**：温暖可爱的卡通风格
- **主色调**：
  - 主色：#FF9A8B（珊瑚粉）
  - 辅助色：#A8E6CF（薄荷绿）
  - 强调色：#FFE66D（阳光黄）
  - 背景色：#FFF8F0（奶油白）
- **圆角设计**：大圆角（20-30px）营造友好感
- **字体**：圆润可爱的中文字体（如：Noto Sans SC Bold）
- **图标风格**：可爱emoji风格

### 4.2 页面设计

#### 首页
| 模块 | 设计描述 |
|------|----------|
| Logo区域 | 居中显示，顶部40%区域 |
| 应用标题 | 大字"动物乐园"，圆润字体 |
| 功能入口 | 圆角卡片，带图标和动效 |

#### 认识动物页面
| 模块 | 设计描述 |
|------|----------|
| 分类导航 | 横向标签栏，当前选中高亮 |
| 动物卡片 | 网格布局，卡片带阴影和hover效果 |
| 动物图片 | 圆角矩形，400x400px |
| 动物名称 | 图片下方，大字显示 |

#### 动物详情弹窗
| 模块 | 设计描述 |
|------|----------|
| 图片展示 | 全屏背景模糊，大图居中 |
| 语音播放 | 文字+动画效果 |
| 关闭按钮 | 右上角圆形按钮 |

### 4.3 响应式设计
- **平板优先**：1024px以上
- **触摸优化**：点击区域至少60x60px
- **动画反馈**：点击有弹性效果
